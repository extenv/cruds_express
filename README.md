> Simple CRUDS
## Tech

 1. NodeJS (18.3.0)
 2. ExpressJS (4.18.1)
 3. Mysql (2.18.1)
 4. Tailwindcss (3.0.24)

## How to use
1). Clone Repository
```
git clone https://github.com/extenv/cruds_express.git
```
2). Move to project
```
cd cruds_express
```
3). Install Packages
```
yarn install
```
  **or**
```
npm install
```
4). Import database
```
localhost/phpmyadmin > import > select db.sql > done
``` 
5). Running
```
yarn dev
```
  **or**
```
npm dev
```
